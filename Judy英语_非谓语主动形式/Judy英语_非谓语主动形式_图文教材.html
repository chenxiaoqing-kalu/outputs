第一部分 › 概念建构

第一部分：非谓语主动形式的语境判断——生理动作算主动还是被动？

在英语语法中，非谓语主动形式的难点往往不在于形式本身，而是在于一些特殊的语境和词的判断。当你不知道某个动作到底是主动还是被动时，你需要进行大量的练习、犯错、复盘与总结。让我们先从一个贴近生活的情境入手，来判断一下生理本能反应到底该用什么语态。

[图：屏幕上书写着绿色的“非谓语的主动形式”大标题，右侧有红色手写词 tremble 和 trembling]

打个比方："将语法中的“主动状态”比作“人体自然的生理反应”，因为它们共享“自身发出、无需外界施加”的特征"。为了更清楚地理解这一点，来看一个例子：

第一部分 › 示例演示
【语境判断】寒冷情境下的生理反应动作状态判断

大雪天一个无家可归的人缩在角落里面，他的手就不停地在抖 (tremble)。这里的“抖”，如果用非谓语的话，应该是主动还是被动？

分析步骤：

  • 确认动作的发出者：手。
  • 判断动作的性质：人在极寒或紧张状态下（比如紧张时腿抖、冒汗），手抖是不受大脑控制的。但这并不是他在故意搞笑作怪，动作依然是由手自身发出的（它是“动”的）。
  • 得出结论：自身发出的动作即属于主动状态。
  • 变形：应用非谓语主动形式 trembling。

判定结果：主动状态（trembling）

从这个例子可以看出，即使动作不受主观意识控制，只要它是从物理源头自身发出的，就依然是主动语态。简单来说："看到“生理本能（发抖、冒汗）”，立刻想到“自己发出的动作，选主动（doing）”"。

需要注意的是："不要以为动作不受大脑控制就是被动，错误的做法是选 done，正确做法是认准动作发出源头选 doing"。

第二部分：非谓语基本主动形式与时间逻辑——基础知识分类复习

理解了主动状态的判断标准后，我们来复习一下非谓语动词的三种基本主动形式及其与主句之间的时间先后逻辑。

[图：电子白板左侧依次列出红色的 doing, having done, to do，右侧用红色笔迹标注着“伴随”、“先发生/完成”、“后发生”]

学习基础知识，要像管理生活环境一样清晰。可以这样理解："将“基础知识在脑中的管理”比作“整理房间”，因为它们共享“分类存放、条理清晰才好找”的特征"。不要让知识点在脑子里面打架，它要非常规整地待在它该待的地方。下面通过梳理来明确这三种形式的时间逻辑：

第二部分 › 示例梳理
【形式归类】非谓语动词基本主动形式的时间分类

非谓语的主动形式复习，区分 doing, having done, to do 的时间发生关系。

分类梳理：

  • doing：表示“伴随”状态，与主句动作在时间上无先后顺序。
  • having done：表示动作在主句动作之前“先发生”，或者表示该动作“已做完（完成）”。
  • to do：大家最熟悉的用法是作目的状语，而在时间发生逻辑上，它表示“后发生”。

对应关系：doing (伴随/无先后)；having done (先发生/完成)；to do (后发生)

有了前面的铺垫，我们在做题时就能建立快速反应。记住："看到“表伴随”，立刻想到“doing，时间无先后”"；同时："看到“having done”，立刻想到“先发生或已完成”"；另外："看到“表目的”，立刻想到“to do，时间后发生”"。

常见的误区是混淆它们的时间轴位置："不要把基础概念记混，正确做法是认准形式对应的时间轴位置"。

第三部分：固定搭配中的非谓语进行时应用——pretend to be doing 怎么推导？

当基本的形式搞清楚后，我们遇到固定搭配就会简单很多。有的时候，固定搭配的句法要求与实际语境的时态要求会同时出现，这时我们需要对非谓语结构进行叠加。

第三部分 › 模型推演
【模型推演】pretend 与不定式进行时的固定搭配

如果我想表达“假装正在做某事”，使用 pretend 这个词，应该如何写这个固定搭配？

推导步骤：

  • 分析固定搭配：pretend 后面必须加不定式 to do，构成 pretend to。
  • 分析时态语义：根据语境，我还需要表达“正在做”的意思，即进行时 be doing。
  • 结构合并：我又有这个 to，又有表示“正在”的意思，那就只能将两者合并，写出 to be doing 结构。
  • 得出完整结构：将两者拼凑，最终得到 pretend to be doing。

正确写法：pretend to be doing

这个分析揭示了一个关键点：复杂的语法结构往往是由简单的固定要求叠加而成的。一个实用的口诀是："看到“固定搭配要求 to + 语义要求正在”，立刻想到“推导出 to be doing”"；或者直接记住："看到“假装正在”，立刻想到“pretend to be doing”"。

容易犯的错误是漏掉介词："只写 pretend doing 是错的，正确做法是遵从 pretend to 的固定搭配原则"。

第四部分 › 语境分析
【语境分析】only to do 意料之外结果的语境分析

理解 only to find 的语境：他在战场上打了很多年的仗，回到家发现他老婆跑了；你拼尽全力跑到公交车站，结果只发现它已经走了。

分析步骤：

  • 给出主句的努力动作：比如在战场打仗多年试图回家团聚，或者拼尽全力跑到公交车站。
  • 点明隐藏的前提：这种努力的前提是你心里有正向的预期（觉得万一能赶上车，或者满心期待见到家人）。如果你知道肯定赶不上，车肯定已经走了，你就不会跑了。
  • 给出客观事实：事与愿违，老婆离开了家，或者车已经走了。
  • 结论：由于结果与强烈的正向预期产生了巨大反差，这种意料之外的遗憾和挫败感，就需要使用 only to do 结构来强调。

结构用法：使用 only to find / only to do 表达意料之外的遗憾结果。

总结一下这道题的核心逻辑：这也是只有在你深刻理解了语境前提后才能做对的题目。可以这样理解："将“意料之外”比作“跑到车站车刚走”，因为它们共享“努力与回报截然相反”的特征"。简单来说："看到“拼尽全力却事与愿违的语境”，立刻想到“only to do 结构”"。

需要注意的是："不要以为所有结果状语都用 doing，遇到强烈转折和意料之外的结果，正确做法是使用 only to do"。

本课总结

本课总结

核心要点

  • 非谓语动作的主动状态属性（不受大脑控制的生理反应只要由自身发出也是主动）。
  • 非谓语动词的三种主动形式及其对应时间特征（doing伴随，having done先发生，to do后发生）。
  • to be doing 结构的使用规则（兼顾固定搭配 to 与进行时 be doing）。
  • only to do 结构表意料之外结果的固定搭配逻辑。

易错提醒

  • 不要以为动作不受大脑控制就是被动，错误的做法是选 done，正确做法是认准动作发出源头选 doing
  • 不要把基础概念记混，正确做法是认准形式对应的时间轴位置
  • 只写 pretend doing 是错的，正确做法是遵从 pretend to 的固定搭配原则
  • 不要以为所有结果状语都用 doing，遇到强烈转折和意料之外的结果，正确做法是使用 only to do

知识应用范围

这些知识可以广泛迁移应用于判断各种看似被动但实则为生理本能（如流汗、落叶等）的主动语态选择；适用于所有需依据主句从句时间顺序选择非谓语填空的语法题；适用于 pretend to, happen to 等必接不定式的进行时填空；以及阅读理解中长难句语义转折的推断。
