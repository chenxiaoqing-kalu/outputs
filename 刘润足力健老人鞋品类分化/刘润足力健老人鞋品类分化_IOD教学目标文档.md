## Part B：示例/例题/习题（Worked Examples）

### EX-01：运动鞋→球鞋→篮球鞋的品类分化演变

| 字段 | 内容 |
|------|------|
| **示例 ID** | EX-01 |
| **来源章节/位置** | 第1节：什么是"品类分化"？ |
| **示例类型** | 概念演示 |

#### 题目原文
以"运动鞋"的历史演变为例：作者小时候"运动鞋"是一个统一品类；今天去商场已无法买到笼统的"运动鞋"，而是被细分为球鞋、跑鞋、旱冰鞋、登山鞋、舞鞋、旅游鞋等。进一步，"球鞋"又分化为足球鞋、篮球鞋、网球鞋、羽毛球鞋、高尔夫球鞋等。问：这个现象说明了什么？品类分化是如何发生的？

#### 求解目标
理解品类分化的定义，理解"认知隔离"的判断标准。

#### 解题步骤
1. 随着中国经济增长，市场规模扩大，"球鞋"等非主流需求的销量不断增加；
2. 当"球鞋"销量大到足以养活至少一家专业公司时，它从"运动鞋"树上独立出来；
3. 完成认知隔离的标志：消费者穿跑鞋打篮球会被人鄙视——说明两者在认知中已被明确区分；
4. 认知隔离完成 → 品类分化完成；
5. 同理，"球鞋"再次分化出篮球鞋、足球鞋等子品类，过程循环往复。

#### 正确答案
品类分化的本质：随市场规模扩大，非主流需求达到经济可行规模，产生认知隔离，形成新品类。新趋势带来新品类，新品类带来新品牌。

#### 结构化条件
| 条件编号 | 条件内容 | 条件类型 |
|----------|----------|----------|
| C1 | 市场整体规模持续扩大 | 初始状态 |
| C2 | 某细分需求（如球鞋）销量持续增长 | 过程变化 |
| C3 | 细分需求规模达到"可养活一家专业公司"的门槛 | 约束条件 |
| C4 | 消费者形成明确的品类区分意识（穿错被鄙视） | 终止状态 |

#### 学科结构（用于生成变式题）
| 维度 | 描述 |
|------|------|
| **核心逻辑关系** | 市场规模扩大 → 细分需求可行 → 认知隔离 → 品类独立 |
| **关键操作类型** | 市场分层分析、品类树结构拆解 |
| **可变参数** | 具体品类名称、所在行业 |
| **结构不变量** | "规模门槛 + 认知隔离"是品类分化的两个必要条件 |

#### 题型特征标签
`概念演绎` `市场结构分析` `品类树拆解` `历史演变案例`

### EX-02：足力健调研4000老人脚型，发现12个专属痛点

| 字段 | 内容 |
|------|------|
| **示例 ID** | EX-02 |
| **来源章节/位置** | 第2节：老人鞋品类识别与调研 |
| **示例类型** | 模型分析 |

#### 题目原文
足力健如何证明"老人鞋"确实是独立于普通鞋的新品类？调研了约4000个老人的脚，找到老人穿鞋特有的12个问题。重点讲解三大问题：第一大问题：鞋子挤脚（67.4%老人脚变宽，原因：足弓塌陷+拇指外翻，本质是形状问题而非大小问题）；第二大问题：磨脚（52.1%老人脚后跟磨损最重，皮下脂肪流失导致干裂）；第三大问题：鞋底打滑（55.17%老人因穿鞋不当摔倒，腿无力+鞋底硬+触地面积小）。

#### 求解目标
说明如何通过实地调研发现特定人群的专属痛点，以支撑品类分化的合理性。

#### 解题步骤
1. 确定调研对象：研究老人的脚（而非老人的意见）；
2. 实地调研：去公园、商场、家访，测量约4000个老人的脚；
3. 归纳专属痛点：找到12个老年人穿鞋才会遇到的问题；
4. 针对痛点设计解决方案：挤脚→单独设计宽鞋楦；磨脚→设计"U型跟"包裹脚后跟；打滑→扩大鞋底触地面积、防滑鞋底、防撞鞋头弧度；
5. 验证：3天复购率高 → 品质兑现承诺 → 认知隔离确立。

#### 正确答案
通过实地调研发现三大专属痛点并针对性解决，完成了老人鞋与普通鞋的"认知隔离"，验证指标为高复购率（3天复购率）。

#### 结构化条件
| 条件编号 | 条件内容 | 条件类型 |
|----------|----------|----------|
| C1 | 2014年搜索"老人鞋"无任何品牌 | 初始状态 |
| C2 | 测量约4000个老人脚型 | 过程变化 |
| C3 | 67.4%老人脚变宽（形状问题） | 约束条件 |
| C4 | 52.1%脚后跟磨损最重 | 约束条件 |
| C5 | 55.17%因穿鞋不当摔倒 | 约束条件 |
| C6 | 3天高复购率作为品质验证指标 | 终止状态 |

#### 学科结构（用于生成变式题）
| 维度 | 描述 |
|------|------|
| **核心逻辑关系** | 特定人群生理/行为特征 → 专属痛点 → 差异化产品设计 → 认知隔离建立 |
| **关键操作类型** | 实地调研、痛点结构化拆解、解决方案映射 |
| **可变参数** | 目标人群（老人→孕妇/儿童/运动员等）、具体痛点数量与内容 |
| **结构不变量** | 必须找到"该人群专属、其他人群不存在"的痛点，才能支撑品类独立 |

#### 题型特征标签
`用户调研方法` `痛点发现与分析` `品类差异化论证` `数据驱动决策`

### EX-03：足力健央视戏曲频道广告，实现1:6的ROI

| 字段 | 内容 |
|------|------|
| **示例 ID** | EX-03 |
| **来源章节/位置** | 第3节：先找红利 |
| **示例类型** | 方法示范 |

#### 题目原文
2016年互联网"品效合一"广告兴起（Angelababy代言美宝莲一场PR活动卖出2万只口红，价值130万元）。足力健面对"老年人不上网"的困境，张京康将"品效合一"逻辑迁移到老人看的央视戏曲频道：制作3分钟广告，投放约100万，广告中多留一个电话号码。结果：电话被打爆，实现1:6的投资回报率（ROI）。2017年投入1.2亿广告费，换回近8亿销售，当年总收入20亿。

#### 求解目标
说明如何识别并利用"渠道红利"（短暂供需失衡），实现低成本快速增长。

#### 解题步骤
1. 识别互联网品效合一红利的本质：广告触达+即时转化，ROI显著高于纯品牌广告；
2. 洞察老年人群体媒介习惯：不上网，但看电视（尤其戏曲频道）；
3. 将"品效合一"逻辑迁移到电视渠道：广告 + 电话下单；
4. 小规模测试：投放100万，验证ROI；
5. 验证成功（ROI=1:6）→ 规模复制：2017年投入1.2亿；
6. 预判红利消失时机：竞争者涌入 → ROI回归1:1 → 停止依赖红利，转入效率竞争。

#### 正确答案
识别红利（短暂供需失衡）→ 小规模验证 → 规模化复制 → 红利消退后及时转型。红利本质是时间窗口，不可依赖，必须提前布局效率壁垒。

#### 结构化条件
| 条件编号 | 条件内容 | 条件类型 |
|----------|----------|----------|
| C1 | 互联网品效合一红利出现（2016年） | 初始状态 |
| C2 | 足力健目标用户（老人）不上网，主要看电视 | 约束条件 |
| C3 | 首次测试：投放央视戏曲频道约100万 | 过程变化 |
| C4 | 实现1:6的ROI | 终止状态 |
| C5 | 2017年规模化：1.2亿广告→近8亿销售→总收入20亿 | 过程变化 |
| C6 | 竞争者进入后ROI回归1:1，红利消失 | 终止状态 |

#### 学科结构（用于生成变式题）
| 维度 | 描述 |
|------|------|
| **核心逻辑关系** | 渠道红利（供需失衡）→ 迁移匹配 → 小测验证 → 规模复制 → 时机判断退出 |
| **关键操作类型** | 红利识别、跨渠道逻辑迁移、ROI验证、规模化决策 |
| **可变参数** | 渠道类型（电视→短视频→社群等）、目标人群媒介习惯 |
| **结构不变量** | 红利必然消失，规模化窗口期有限，需同步布局效率壁垒 |

### EX-04：自建工厂效率攻坚，实现2.5倍定倍率与90%+原价率

| 字段 | 内容 |
|------|------|
| **示例 ID** | EX-04 |
| **来源章节/位置** | 第3节：再抓效率 |
| **示例类型** | 模型分析 |

#### 题目原文
红利消退后，足力健面临代工厂管理失控问题（偷码节省成本、换料、极端情况退货率15%）。2017年红利顶峰时决定自建工厂，从纱线到鞋帮到鞋楦全部自制。最终结果：生产成本降低15%；库存天数大幅缩短，原价率达90%以上；定倍率仅2.5倍（行业水平5-10倍，互联网电商平均3倍）。2019年销售30多亿，利润约2亿。

#### 求解目标
说明如何通过效率提升（品质+成本+定倍率）在红利消退后建立可持续竞争壁垒。

#### 解题步骤
1. 识别代工厂管理风险：偷码（43码→42.5码，省1.5元/双×100万双=150万元）、换料、品质不稳定；
2. 决策自建工厂：在红利顶峰期投入，用红利利润支持基础设施建设；
3. 效率攻坚动作：全链路自制（纱线→鞋帮→鞋楦）；
4. 成果：生产成本降低15%；返单速度加快→库存天数缩短→原价率90%+；
5. 最终定倍率2.5倍（远低于行业5-10倍），成为护城河；
6. 逻辑：定倍率低 → 同等售价下毛利空间更小 → 竞争者难以匹配 → 壁垒形成。

#### 正确答案
自建工厂实现2.5倍定倍率（行业最低），是效率竞争阶段的核心护城河。原价率90%+减少了打折损耗，提升了实际盈利能力。

#### 结构化条件
| 条件编号 | 条件内容 | 条件类型 |
|----------|----------|----------|
| C1 | 代工厂管理风险暴露（退货率15%极端情况） | 初始状态 |
| C2 | 2017年红利顶峰，决定自建工厂 | 过程变化 |
| C3 | 全链路自制（纱线、鞋帮、鞋楦） | 过程变化 |
| C4 | 生产成本降低15% | 终止状态 |
| C5 | 原价率达90%以上 | 终止状态 |
| C6 | 定倍率2.5倍（行业5-10倍，电商均值3倍） | 终止状态 |

#### 学科结构（用于生成变式题）
| 维度 | 描述 |
|------|------|
| **核心逻辑关系** | 红利消退 → 效率竞争 → 降成本/提品质/缩库存 → 定倍率优势 → 护城河 |
| **关键操作类型** | 供应链垂直整合、库存周转率优化、定倍率管理 |
| **可变参数** | 行业类型、垂直整合程度、定倍率基准值 |
| **结构不变量** | 效率壁垒的本质是让竞争者难以低成本复制的系统性能力 |

#### 题型特征标签
`供应链效率优化` `定倍率管理` `竞争壁垒构建` `红利→效率转型`
